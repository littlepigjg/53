## 1. 产品概述

内网文档协作审阅工具，支持上传 Word/Markdown 文档并生成分享链接，审阅者可在浏览器中逐段添加批注和建议修改，文档所有者可后台查看汇总意见并接受或拒绝每条建议。

- **核心目标**：提供轻量级、类 Word 审阅模式的在线协作审阅体验
- **目标用户**：企业/团队内部文档作者、审阅者、项目经理
- **解决痛点**：传统邮件审阅流程繁琐、Word 审阅模式依赖本地软件、意见分散难以汇总

## 2. 核心功能

### 2.1 用户角色

| 角色 | 登录方式 | 核心权限 |
|------|----------|----------|
| 文档所有者 | 简单口令/默认登录 | 上传文档、生成分享链接、查看审阅汇总、接受/拒绝修改建议、导出最终文档 |
| 审阅者 | 链接直接访问（可选口令保护） | 阅读文档、逐段添加批注、提交修改建议 |

### 2.2 功能模块

1. **文档上传页**：拖拽上传 Word(.docx)/Markdown，文档列表展示
2. **文档审阅页**：左侧文档阅读器（段落高亮），右侧审阅面板（批注列表+建议编辑）
3. **管理后台页**：审阅意见汇总、按段落/审阅者筛选、接受/拒绝操作面板
4. **分享链接管理**：生成带 token 的分享 URL，支持过期时间和访问口令

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 文档上传页 | 上传区域 | 拖拽或点击上传 .docx/.md 文件，显示上传进度 |
| 文档上传页 | 文档列表 | 展示已上传文档（标题、创建时间、审阅人数、批注数），支持生成/复制分享链接、进入管理后台、删除 |
| 文档审阅页 | 文档阅读器 | Markdown 渲染、段落级拆分、鼠标悬停高亮、点击段落选中、已批注段落标记 |
| 文档审阅页 | 审阅面板 | 审阅者信息输入、当前段落批注列表、新建批注（批注类型：意见/建议修改）、建议修改的原文对比 |
| 管理后台页 | 汇总视图 | 按段落展示所有批注和建议、统计面板（批注总数、待处理数、已接受/已拒绝）、筛选器（按审阅者/类型/状态） |
| 管理后台页 | 操作面板 | 逐条接受/拒绝建议、批量操作、添加备注、导出最终合并文档 |

## 3. 核心流程

**文档审阅主流程**：
文档所有者上传 Word/Markdown → 系统解析文档并拆分为结构化段落 → 生成带 Token 的分享链接 → 审阅者通过链接打开 → 点击段落添加批注或建议修改 → 批注数据实时保存为 JSON → 所有者进入后台查看汇总 → 逐条接受/拒绝建议 → 导出合并后文档。

```mermaid
flowchart LR
    A["文档所有者"] --> B["上传 Word/Markdown"]
    B --> C["解析文档并拆分为段落"]
    C --> D["生成分享链接"]
    D --> E["分享给审阅者"]
    E --> F["审阅者打开链接"]
    F --> G["点击段落添加批注/建议"]
    G --> H["保存至 JSON 文件"]
    H --> I["所有者查看汇总"]
    I --> J["接受/拒绝每条建议"]
    J --> K["导出最终文档"]
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深蓝 #1e3a5f（专业、稳重），辅色：琥珀 #d97706（标注、强调）
- **按钮样式**：圆角 6px，轻微阴影，hover 时背景加深
- **字体**：正文使用 "Noto Serif SC"（宋体风格，适合文档阅读），UI 交互使用 "Inter"
- **布局风格**：三栏式布局（导航栏 + 文档区 + 审阅面板），卡片式组件，充足留白
- **图标风格**：Lucide 线性图标，简洁一致

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|-----------|-------------|-------------|
| 文档上传页 | 上传区域 | 虚线边框拖拽区，大尺寸上传图标，文件格式提示 |
| 文档上传页 | 文档列表 | 表格布局，每行显示状态徽标，行 hover 显示操作按钮 |
| 文档审阅页 | 文档阅读器 | 最大宽度 720px 居中，行高 1.8，段落左侧批注标记条，选中段落左侧琥珀色竖线 |
| 文档审阅页 | 审阅面板 | 固定宽度 360px 右侧边栏，白色背景，分段卡片，输入框带类型切换标签 |
| 管理后台页 | 汇总视图 | 顶部统计卡片网格，筛选器工具栏，双栏布局（文档预览 + 意见列表） |
| 管理后台页 | 操作面板 | 每条意见右下角操作按钮组（接受/拒绝），建议修改显示原文与建议对比框 |

### 4.3 响应式

桌面端优先设计，审阅页和后台页为双栏/三栏布局；平板端自适应折叠侧栏；移动端以单列阅读为主，审阅面板以底部抽屉形式呈现。
